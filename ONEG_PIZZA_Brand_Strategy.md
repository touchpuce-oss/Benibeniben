# 1. BRAND STRATEGY SUMMARY

## Brand Essence

**ONEG PIZZA** is not a food truck. It is a **mobile culinary atelier**—where Neapolitan tradition meets live performance artistry. Every event becomes a memory. Every pizza is a statement of craft.

### Core Brand Pillars

| Pillar | Expression |
|--------|------------|
| **Artisan Mastery** | 48-hour sourdough fermentation, certified Neapolitan technique, hand-stretched dough |
| **Live Theater** | The firewood oven as centerpiece—flames, aroma, the dance of the pizzaiolo |
| **Premium Mobility** | From intimate garden parties to 200-guest corporate galas—seamless execution |
| **Authentic Warmth** | Refined without pretension. Luxury that feels like an invitation, not an exclusion |

### Brand Archetype: **The Artisan + The Magician**
- **Artisan**: Mastery, tradition, attention to detail
- **Magician**: Transformation, experience, creating moments of wonder

### Brand Voice
- **Confident** but never arrogant
- **Warm** but never casual
- **Precise** but never cold
- **Passionate** but never effusive

### What We Are NOT
- ❌ Street food quick-bite positioning
- ❌ Price-driven commodity
- ❌ Overly playful or emoji-heavy
- ❌ Rustic-to-the-point-of-unpolished

---

## Value Proposition

> **"We bring the soul of Naples to your celebration—fire, fermentation, and the art of live cooking."**

### For Event Clients:
- **The Experience**: Your guests don't just eat—they witness craft in motion
- **The Flexibility**: Firewood oven outdoors, professional electric indoors—any venue, any season
- **The Certainty**: Seamless execution from 30 to 200+ guests, handled with precision

### For Weekly Customers (Gagny):
- **The Authenticity**: True Neapolitan sourdough, made with time-honored methods
- **The Connection**: Direct WhatsApp access, personal service, community rooted

# 3. VISUAL DIRECTION CONCEPT

## Design Philosophy: **"Warm Minimalism"**

A sophisticated balance between artisan authenticity and refined luxury. Think: the warmth of a Naples pizzeria meets the editorial elegance of a design magazine.

### Color Palette

```
Primary Palette:
┌────────────────────────────────────────────────────────┐
│  DEEP CHARCOAL      #1A1A1A    (Backgrounds, text)    │
│  WARM CREAM         #F5F0E8    (Primary background)    │
│  TERRACOTTA         #C45C3E    (Primary accent, CTAs)  │
└────────────────────────────────────────────────────────┘

Secondary Palette:
┌────────────────────────────────────────────────────────┐
│  FIRE AMBER         #E87A3D    (Hover states, flames) │
│  OLIVE GREEN        #5A6B4C    (Natural accent)       │
│  BURNT SIENNA       #8B4513    (Wood, earth tones)    │
│  SOFT GREY          #E8E4DC    (Borders, dividers)    │
└────────────────────────────────────────────────────────┘

Usage:
• Background: Warm cream (#F5F0E8) for main, Deep charcoal (#1A1A1A) for CTAs/footer
• Text: Deep charcoal on cream, cream on charcoal
• CTAs: Terracotta with cream text
• Accents: Fire amber for hover, olive for natural elements
```

### Typography Pairing

```
HEADINGS: Playfair Display (or Cormorant Garamond)
─────────
• Elegant serif with Italian heritage
• Weight: 600–700 for headlines
• Style: Timeless, editorial, confident

BODY: Inter (or Source Sans Pro)
─────
• Clean, highly legible sans-serif
• Weight: 400–500 for body, 600 for emphasis
• Style: Modern, approachable, precise

ACCENT: Cormorant Garamond Italic
──────
• For quotes, taglines, subtle emphasis
• Weight: 400 italic
• Style: Refined, personal, warm
```

### Typography Scale

| Element | Font | Size (Desktop) | Size (Mobile) | Weight |
|---------|------|----------------|---------------|--------|
| H1 (Hero) | Playfair Display | 64px | 40px | 700 |
| H2 (Section) | Playfair Display | 42px | 32px | 600 |
| H3 (Cards) | Playfair Display | 28px | 24px | 600 |
| Body | Inter | 18px | 16px | 400 |
| Caption | Inter | 14px | 14px | 400 |
| CTA | Inter | 16px | 16px | 600 |

### Image Treatment Style

**Photography Direction:**
- **Warm, natural lighting**—golden hour, fire glow, candlelight
- **Shallow depth of field**—focus on texture (bubbles in dough, char on crust)
- **Authentic moments**—hands at work, flour in the air, genuine expressions
- **Rich, warm tones**—enhance amber, terracotta, cream palette
- **Editorial composition**—think Kinfolk, Cereal Magazine aesthetic

**Avoid:**
- Harsh flash photography
- Overly saturated, "food blog" styling
- Stock-looking images
- Cluttered compositions

### Button Design Language

```
Primary CTA:
┌─────────────────────────┐
│  Background: Terracotta │
│  Text: Cream, uppercase │
│  Padding: 18px 40px     │
│  Border-radius: 0px     │
│  Letter-spacing: 1px    │
│  Hover: Darken 10%      │
└─────────────────────────┘

Secondary CTA:
┌─────────────────────────┐
│  Background: Transparent│
│  Border: 2px Charcoal   │
│  Text: Charcoal         │
│  Hover: Charcoal bg     │
│         Cream text      │
└─────────────────────────┘

WhatsApp CTA:
┌─────────────────────────┐
│  Background: #25D366    │
│  Text: White            │
│  Icon: WhatsApp logo    │
│  Position: Sticky (mob) │
└─────────────────────────┘
```

### Section Spacing Philosophy

- **Generous whitespace**—luxury needs room to breathe
- **Section padding**: 120px vertical (desktop), 80px (mobile)
- **Content max-width**: 1200px, centered
- **Asymmetric balance**—avoid perfect symmetry for visual interest
- **Full-bleed moments**—hero, gallery, final CTA break the container

### Animation & Micro-interactions

| Element | Animation | Timing |
|---------|-----------|--------|
| Page load | Fade in + subtle Y translate | 600ms, ease-out |
| Scroll reveal | Fade up (20px) | 400ms, stagger 100ms |
| Button hover | Scale 1.02, shadow | 200ms |
| Card hover | Lift (translateY -4px), shadow | 300ms |
| Gallery | Masonry layout, lightbox zoom | 300ms |
| Nav scroll | Background opacity 0→1 | 200ms |

**No:** Parallax (distracting), auto-playing carousels (annoying), excessive motion (cheapens)

### Mobile-First Behavior

- **Sticky WhatsApp CTA** at bottom—always accessible
- **Hamburger menu** with smooth overlay
- **Touch-friendly** targets (min 44px)
- **Optimized images**—WebP, lazy loading
- **Single-column layout**—clean hierarchy
- **Click-to-call** for phone numbers

### Sticky CTA Concept

**Desktop:**
- Fixed navigation with "Book Now" button always visible
- WhatsApp icon in nav

**Mobile:**
- Bottom-fixed bar with:
  - "Book Event" (primary)
  - WhatsApp icon (secondary)
- Collapses on scroll down, reappears on scroll up

# 4. CONVERSION FUNNEL STRATEGY

## Primary Goal: Event Bookings
## Secondary Goal: WhatsApp Engagement

### CTA Placement Strategy

| Location | CTA Type | Wording |
|----------|----------|---------|
| Hero | Primary | "Book Your Event" |
| Hero | Secondary | "View Our Menu" |
| Experience Section | Primary | "Plan Your Setup" |
| Event Packages | Primary | "Request Your Quote" |
| Menu Section | Secondary | "Get Full Menu on WhatsApp" |
| Gallery | Tertiary | "See More on Instagram" |
| FAQ | Primary | "Still Have Questions? Ask on WhatsApp" |
| Final CTA | Primary | "Book Your Event" |
| Sticky (mobile) | Primary | "Book Now" |

### Event Booking CTA Wording

**High-Conversion Phrases:**
- ✅ "Book Your Event" (direct, confident)
- ✅ "Request Your Personalized Quote" (premium, tailored)
- ✅ "Reserve Your Date" (scarcity, urgency)
- ✅ "Plan Your Experience" (emotional, consultative)

**Avoid:**
- ❌ "Contact Us" (too generic)
- ❌ "Get a Price" (price-focused, cheapens)
- ❌ "Order" (transactional, not experiential)

### WhatsApp Conversion Strategy

**Pre-filled Message Strategy:**

Link: `https://wa.me/33XXXXXXXXX?text=...`

**Message Templates:**

1. **General Inquiry:**
   ```
   Bonjour ONEG PIZZA,
   
   I'm interested in booking your pizza experience for an event.
   
   • Date: [Your date]
   • Number of guests: [Approximate]
   • Location: [Your location]
   • Type of event: [Wedding / Corporate / Private]
   
   Could you please send me your availability and packages?
   
   Thank you!
   ```

2. **Menu Request:**
   ```
   Bonjour ONEG PIZZA,
   
   I'd love to receive your full menu and weekly location in Gagny.
   
   Thank you!
   ```

**Why this works:**
- Lowers friction (pre-structured inquiry)
- Gathers key qualifying info
- Professional tone sets expectation
- French greeting + English body (bilingual flexibility)

### Urgency & Scarcity Tactics

| Tactic | Implementation |
|--------|----------------|
| **Seasonal scarcity** | "Peak season (May–Sept) books 3–6 months ahead" |
| **Date limitation** | "Limited dates remaining for 2025" |
| **Social proof count** | "200+ events catered" |
| **Response time** | "Response within 24 hours" |

### Elevating Perceived Value

**Language Choices:**
- "Experience" not "service"
- "Catering" not "food truck"
- "Reserve" not "book"
- "Atelier" not "kitchen"
- "Pizzaiolo" not "cook"

**Visual Cues:**
- No prices on homepage (packages only)
- "Starting from" language, never "only"
- Full words, no abbreviations
- Proper accents (pizzaiolo, Neapolitan)

**Trust Signals:**
- Event count (200+)
- Certification mention
- Client names + venues (when permitted)
- Professional photography
- Clean, uncluttered design

### Conversion Flow

```
Visitor arrives
      ↓
Hero: Emotional hook + dual CTAs
      ↓
Philosophy: Build trust (craft, expertise)
      ↓
Experience: Show versatility (outdoor/indoor)
      ↓
Packages: Present options (choice architecture)
      ↓
Menu: Showcase quality (desire building)
      ↓
Social Proof: Remove doubt (testimonials)
      ↓
Gallery: Visual confirmation (aspiration)
      ↓
FAQ: Handle objections (confidence)
      ↓
Final CTA: Close the loop (action)
```

# SUMMARY

**ONEG PIZZA** is positioned as a premium mobile culinary experience—not a food truck. The website must reflect this through:

1. **Warm Minimalism** design aesthetic—sophisticated yet approachable
2. **Experience-first messaging**—focus on emotion, not just food
3. **Strategic CTA placement**—guide visitors toward event bookings
4. **WhatsApp integration**—frictionless conversion channel
5. **SEO optimization**—capture local French event catering searches
6. **Social proof elevation**—testimonials that reinforce premium positioning

The result: A website that converts visitors into high-value event clients while building brand desirability in the Gagny and Île-de-France market.
